This project analyzes customer behavior and purchasing patterns to understand the factors influencing product buying decisions. The analysis is performed using Pandas for data cleaning and exploration, PostgreSQL for structured querying and advanced SQL analysis, and Power BI for interactive dashboards and visual insights.

The project focuses on identifying:

Purchase trends and customer preferences

Repeat buying behavior and subscription likelihood

Category-wise and item-wise performance

Insights that can support data-driven business decisions

Tech Stack:

Python (Pandas)

PostgreSQL (SQL)

Power BI (Data Visualization)
